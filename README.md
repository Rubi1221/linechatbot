# Smart Travel Assistant Chatbot System

本專案為一套智慧旅遊助理聊天機器人系統，結合即時訊息平台、影像辨識與 AI 文字生成技術，提供使用者景點辨識、旅遊資訊查詢與個人化行程規劃服務。

---

## 功能介紹

- 使用者訊息接收與身份識別
- 圖片輸入：旅遊景點影像辨識
- 文字輸入：
  - 景點／縣市查詢
  - 個人化旅遊行程規劃
  - 歷史互動紀錄查詢
- 使用者資料與查詢紀錄儲存

---

## 系統架構

- 前端／訊息平台：Message API
- 後端服務：Flask
- Webhook：接收與處理使用者訊息
- AI 服務：
  - Microsoft Azure Image Recognition API
  - OpenAI API（ChatGPT）
- 資料庫：SQL Server
